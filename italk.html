Overview

iTalk is an AI-powered language learning web app built on order for a client. The goal was to create a natural conversation experience where users practice speaking with an AI that feels like a real phone call — not a chatbot. MVP delivered in 25 days. The client launched the product under their own brand.

What I Did

AI Conversation Engine

Built the core AI engine using Google Gemini 2.5, adapted to the user's CEFR language level in real time — vocabulary, grammar and speech pace all adjust automatically to the learner.

Voice Sync — The Hard Part

The biggest technical challenge: syncing voice input and AI response into a seamless back-and-forth that feels like a real mobile phone conversation. Web Speech API integrated for both recognition and synthesis.

UI/UX Design & Frontend

Designed and developed the full interface from scratch — mobile-first, dark/light mode, streak tracking and weekly goals. Clean and focused on keeping users in the learning flow.

Multilingual Architecture

Built full i18n support — 50+ learning languages and 5 interface languages including English, Russian, German, Spanish and French. Fully internationalised from day one.

Result

A complete AI product. Delivered in under a month.

25 days from zero to MVP Launched by client under own brand Open source on GitHub
